Que es Odoo?

Odoo es un software de código abierto que actúa como un Sistema de Planificación de Recursos Empresariales (ERP) y un sistema de gestión de relaciones con clientes (CRM). Integra diversas aplicaciones y módulos para gestionar diversas áreas empresariales como contabilidad, ventas, compras, inventario, marketing y recursos humanos, entre otros.
